## プロジェクト説明
### プロジェクト名
Hello World

### 概要
プログラム言語を"Python"として，"Hello World!"を出力する．
出力先はコンソールとなる．

### 特徴
記述言語は"Python"である．

### 動作環境
"Python"が動く環境なら可能．

## インストール方法
特になし．
GitHubからソースコードをダウンロードし，VSCodeなどのプログラムを実行できる環境で実行する．

### 使い方
インストール後，プログラムを編集及び実行可能なアプリケーションを通してプログラムを実行する．実行後，コンソール上に「Hello World!」と出力がされる．
